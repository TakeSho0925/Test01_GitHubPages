# The Beatles ビートルズ
　こんにちは！たけしょーです。今日は僕の大好きなバンドの紹介をしたいと思います！紹介するバンドの名前は**ザ ビートルズ**です！
皆さんも一度は聞いたことがあるのではないでしょうか？世界一有名なバンドです。今日はそんなビートルズのことを皆さんに知っていただけたらと思います！
　

　では行きましょう！！
